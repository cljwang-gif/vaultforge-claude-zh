# 贡献指南

欢迎改进 `vaultforge-claude-zh`。

## 适合贡献的内容

- 更清晰的项目总控协议。
- 更通用的 Obsidian 模板。
- 更稳定的岗位 Skill。
- 适用于小说、研究、产品文档、知识库的示例。
- 文档修订和错别字修正。

## 贡献原则

- 保持框架通用，不绑定私人项目。
- 不加入具体小说设定、私有资料或未授权文本。
- 优先使用小而清晰的文件。
- 指令要可执行、可检查。
- 不复制外部作品的受版权保护文本。

## Pull Request 检查清单

- 变更对多个项目都有用。
- 没有包含私人项目数据。
- 必要文档已更新。
- 模板中使用占位符，而不是具体私有内容。
