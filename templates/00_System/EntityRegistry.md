# EntityRegistry

本索引用于管理地点、组织、概念、物品、能力、系统、规则和反复出现的事件。

## 状态定义

| 状态 | 含义 |
|---|---|
| 临时出现 | 短暂使用 |
| 观察中 | 可能复用 |
| 待升级 | 重要度足够，需要确认 |
| 正式设定 | 长期项目事实 |
| 休眠 | 暂时不用但保留 |
| 退场 | 已完成功能 |
| 废弃 | 不再使用 |
